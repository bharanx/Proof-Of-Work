ProofOfWork — Demo (GitHub Pages)

This is a static copy of the ProofOfWork frontend intended for GitHub Pages. Dynamic features that require the API (peer verification, registration, supply chain, finance) are disabled in this static build.

To run the full app (API + frontend) locally, run the project in Docker as described in the repository README.

Available pages:

  • Interactive frontend (attempts to connect to API) — will only work if an API is hosted and CORS allowed.

Quick instructions

  1. Go to repository Settings → Pages and set Source to the `main` branch / `docs/` folder.
  2. Wait a minute — your site will be available at https://<username>.github.io/<repo>/
